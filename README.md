# Dotfiles Configuration

## Dependencies

- Neovim
- Node-npm
- Stow
- zsh-oh_my_zsh

## Installation

### Cloning the repository

```bash
git clone https://github.com/H0nzrA/Dotfiles.git
cd Dotfiles
```

### Install `Stow`

```bash
sudo apt install stow
# or
sudo snap install stow

```

### Configuration
```bash
stow .
```
